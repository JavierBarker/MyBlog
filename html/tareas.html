Cálculo

Lenguajes con los que se puede programar Android y iOS

La tarea trata de una investigación de lenguajes de programación para hacer aplicaciones móviles, en especial para los dos Sistemas Operativos mas populares para celulares, Hay lenguajes que se usan únicamente para alguno de esos Sistemas Operativos y hay otros que se usan para ambos.

Dibujo

Mockups (Gestor de Hoteles)

Este es un trabajo grupal en la cual Consiste en la creación de las vistas del Proyecto del segundo Bimestre, El Gestor de Hoteles, para ello usamos una aplicación llamada Marvel App, que nos ayuda a poder crear esas vistas. Estas vistas no son interactivas, solamente es visual.

Tecnología

Propuesta de Proyecto

Esta tarea trata de hacer una propuesta con mi grupo, una idea de una aplicación para poder venderla, esto se hace a través de un documento en la que se describe la aplicación detalladamente. Se describe la idea, un Manual de usuario de como se vería la aplicación, el Tiempo Predestinado para realizarlo, Metodologías, etc.

Presentación de Proyecto

Esta tarea trata de exponer Grupalmente la idea planteada en la Propuesta, explicando y resolviendo dudas respecto al proyecto, como la forma en la cual la aplicación tendrá Ganancias, opciones adicionales no explicadas, etc.

Prueba Corta

Este es una evaluación corta relacionado a todos los temas vistos durante el bimestre. La evaluación tratará sobre Nodejs, Express, NPM, MongoDB, Typescript y Angular.

Práctica Supervisada

Exposición #1: Frameworks para aplicaciones hibridas

Esta tarea trata de hacer una Exposición en la que cada grupo explica un Framework para hacer aplicaciones Hibridas (esto quiere decir que van a ser funcionales para Android y iOS). A nuestro Grupo le tocó exponer sobre PhoneGap.

Exposición #2: - Frameworks para Páginas Web

Esta tarea trata de hacer una Exposición en la que cada grupo explica un Framework para hacer Páginas Web, Frameworks especialmente para hacer la parte visual de la aplicación. A nuestro grupo le tocó exponer sobre Symfony.

Taller

Realización de un Blog

Esta tarea trata de hacer un Blog Personal en la cual ire subiendo todas las actividades que vaya haciendo en cada curso con una pequeña descripción de lo que trata la tarea.

Manual de Usuario

Esta tarea trata de hacer un Manual para el Usuario relacionado al proyecto del Bimestre, El Gestor de Hoteles. Para este Manual tomaré de referencia los Mockups de la tarea de Dibujo.

Proyecto Gestor de Hoteles

Esta tarea se trata del Proyecto Final de Bimestre en la cual se tiene que hacer una Aplicación Web hecha en Angular, Nodejs y MongoDB. La aplicación tendra tres Roles (Usuario, Administrador de Hotel y Administrador de la Aplicación), cada uno hará una función diferente.
